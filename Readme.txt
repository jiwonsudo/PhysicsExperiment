이 앱은 고등학교 2학년, 학교 물리 교과시간 내 발표를 위해 제작된 앱입니다.
5개의 페이지로 이루어져 있으며, 메인 메뉴를 통해 각 페이지로 이동할 수 있습니다.
인터랙티브 화면을 만들어 보고 싶어 제작하게 되었습니다.
